# CodeAlpha Internship Tasks

This repository contains my frontend development task submissions for the CodeAlpha May Batch. Both web applications were built with a focus on responsive design, clean UI/UX, and robust offline capabilities.

## 🗂️ Task 1: Flashcard Quiz App
A web application designed for active recall studying.
* **Tech Stack:** HTML5, CSS3, Vanilla JavaScript.
* **Architecture:** Implements a Single Page Application (SPA) structure with smooth CSS 3D flip animations.
* **Data Management:** Fully offline-capable. It utilizes the browser's native `localStorage` API to execute CRUD operations (Create, Read, Update, Delete) without relying on external databases.

## ✨ Task 2: Random Quote Generator
A responsive web application featuring a premium, Apple-inspired Liquid Glass (Glassmorphism) aesthetic.
* **Tech Stack:** HTML5, CSS3, Vanilla JavaScript.
* **Features:** Dynamic DOM manipulation for rendering quotes, and integration with the modern `Navigator Clipboard API` for one-click text copying.
* **Data Management:** Offline-capable. Driven by a local data array structure to ensure zero latency, eliminating the risk of third-party API rate limits or server downtime.
